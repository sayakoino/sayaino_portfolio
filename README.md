# うつ病診断のための機械学習

このプロジェクトでは、音声特徴量を使用してうつ病を診断するための機械学習モデルを開発しました。プロジェクトの主な構成要素は、データ前処理、特徴抽出、モデルの学習、および評価です。

## プロジェクト概要

- **アルゴリズム**: 多層パーセプトロン（MLP）
- **特徴量**: 音声特徴量（MFCC）、性別、その他統計的な特徴
- **モデル**: `scikit-learn`を使用して開発
